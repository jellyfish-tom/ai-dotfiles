description: Use when implementing a Figma design for this project profile.
---

Implement a Figma design in the target repository.

1. Read `.github/instructions/figma-mcp-integration.instructions.md` before implementation.
2. Use Figma MCP: `get_design_context`, then `get_screenshot`.
3. Reuse existing UI primitives and tokens before creating new abstractions.
4. Ask for region, brand, or flow scope when ambiguous.
